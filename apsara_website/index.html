About Apsara 2.0

Apsara 2.0 is a powerful and versatile AI assistant that leverages advanced language models, custom tools, and seamless role transitioning to provide an unparalleled user experience. With its ability to understand complex queries and reason through logical steps, Apsara 2.0 delivers accurate and relevant responses to meet your needs.


  • 🧠 Powered by cutting-edge LLMs for natural language understanding
  • 🛠️ Supports custom user-defined tools for extended functionality
  • 🎭 Seamlessly transitions between chatbot and agent roles

Installation and Usage

  1. Clone the GitHub repository
  2. Run python install.py
  3. Set up environment variables
  4. Run python main.py or python main.py --agent
